# Програмирование картографо-геодезических задач
<p>Каждая ветка данного репозитория представляет собой практическое задание.</p>
<p>
Процесс работы может строится следующим образом:
<li>Установить <a href="https://git-scm.com/downloads">Git</a></li>
<li>Открыв Git Bash, сменить директорию на желаемую с помощью команды <code>cd <i>path</i></code></li>
<li>Скопировать HTTPS репозитория, нажав на <i>Code</i> на странице в GitHub</li>
<li>Клонировать данный репозиторий в выбранную директорию: <code>git clone <i>https</i></code></li>
<li>Выбрать соответствующую ветку с помощью <code>git fetch origin <i>assignment_n</i></code></li>
<li>Переключиться на неё с помощью команды <code>git switch <i>assignment_n</i></code></li>
<li>Унаследоваться от этой ветки с нужным заданием: <code>git checkout -b <i>n_lastname</i></code></li>
<li>В проводнике переместить файлы, являющиеся ответом на задание, в выбранную директорию</li>
<li>Добавить эти файлы с помощью команды <code>git add .</code></li>
<li>Убедиться, что файлы добавлены можно с помощью <code>git status</code></li>
&emsp;&nbsp;Недобавленные файлы отразятся красным, добавленные - зелёным
<li>Внести изменения с помощью <code>git commit -m "<i>n_Фамилия</i>"</code></li>
<li>Отправить изменения из локального репозитория в удалённый: <code>git push origin <i>n_lastname</i></code></li>
<li>В GitHub создать запрос на слияние (pull request) Вашей ветки <i>1_lastname</i> и ветки соответствующего задания <i>assignment_n</i></li>
<li>Баллы за задание или необходимость доработки будут указаны в комметраиях после проверки</li>
</p>
