# Задание 2. Векторные данные. OGR
Основы работы с векторными данными в Python на примере библиотеки OGR:
- Импортировав ogr из osgeo, открыть файл (Open с аргументом 1 для открытия в режиме редактирования), слой (GetLayerByIndex), получить поля атрибутивной таблицы (GetLayerDefn) и для каждого поля вывести название  
- Узнать количество объектов в слое (GetFeatureCount), затем отфильтровать его (SetAttributeFilter) согласно Вашему заданию из таблицы и снова вывести количество (уже отфильтрованных) объектов слоя 
- Создать новое поле (FieldDefn) согласно Вашему заданию из таблицы и добавить его к слою (CreateField), затем для каждого объекта в слое задать значение (SetField) согласно вашему заданию в таблице, а потом присвоить объект слою (SetFeature) 
- Если подразумевается расчёт значения, исходя из пространственной информации, то: объект.geometry().функция() 
- С помощью драйвера GeoJson (GetDriverByName) создать пустой выходной файл (CreateDataSource("/content/drive/MyDrive/2_Фамилия.json")), затем скопировать в него отфильтрованный слой с добавленными значениями (CopyLayer(layer,'2_Фамилия')) и удалить исходный слой, исходный файл, выходной слой и выходной файл (del соответствующие переменные) 
- Если возникают проблемы с сохранением, убедитесь, что на диске нет файла с идентичным названием – это может являться причиной ошибки
