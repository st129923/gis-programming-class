# Задание 3. Растровые данные. GDAL
Основы работы с векторными данными в Python на примере библиотеки GDAL:
- В файле «Задание_3» найти индекс, который Вам необходимо автоматизировано рассчитать в ходе выполнения работы
- Загрузить на диск TIF-файлы необходимых для расчёта каналов, полученные из архива «S2_60m»
- Подключившись в Google Colab к диску, импортировать gdal из osgeo, а также numpy
- Открыть файл каждого из каналов (Open)
- Каждый из открытых каналов считать как массив (ReadAsArray)
- Каждому из полученных массивов задать тип данных float (astype)
- Для всех каналов в архиве значением «нет данных» является 0, соответственно, в массивах необходимо элементам со значением менее 1 присвоить «нет данных» (массив[массив < 1] = numpy.nan)
- С помощью математических операций с массивами рассчитать Ваш индекс (получим новый массив такой же размерности, i-ый элемент которого будет рассчитан по i-ым элементам исходных)
- И снова вернуть значение 0 для «нет данных» (индекс[numpy.isnan(индекс)]=0)
- Вывести минимальное и максимальное значения индекса (numpy.min и max)
- Используя полученный диапазон значений, классифицировать массив (индекс[numpy.where((индекс > значения1) & (индекс <= значения2))] = класс), при этом классы – это значения от 1 до n в зависимости от их количества (выбрать подходящее на Ваш взгляд)
- Определить драйвер «GTiff» (GetDriverByName), с помощью него создать (Create) пустой растровый файл, указав в качестве аргументов путь для сохранения (/content/drive/MyDrive/3_Фамилия.tif), размер по оси X (взять у одного из открытых в начале каналов с помощью RasterXSize), размер по оси Y (аналогично, RasterYSize), количество результирующих каналов (1), тип данных (GDT_Float32)
- Созданному пустому изображению задать геотрансформацию (SetGeoTransform(любой_канал(GetGeoTransform))), проекцию (аналогично: SetProjection / GetProjection)
- Для данного растрового изображения получить первый (и единственный) канал (GetRasterBand), задать ему значение «нет данных», равное 0 (SetNoDataValue) и добавить в этот канал рассчитанный массив, создав индексное изображение (WriteArray), а также очистить переменные, отвечающие за созданный и открытые ранее TIF-файлы, присвоив значение None
- В конце блокнота создать текстовую ячейку и кратко описать полученный результат (открыть результат в QGIS и оценить, получен ли желаемый результат, т.е. видны ли на данном снимке объекты или явления, которые индекс должен был помочь идентифицировать)
